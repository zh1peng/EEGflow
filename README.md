# EEGflow

EEGflow is a MATLAB toolbox for EEG preprocessing and analysis.

## Quick start

1. Add this folder to your MATLAB path.
2. Run `utils/setup.m` or `utils/setup_params.m` to initialize parameters.
3. Use functions in `+prep` for preprocessing and `+analysis` for analysis pipelines.

## Structure

- `+prep`: preprocessing steps and helpers
- `+analysis`: analysis steps and helpers
- `contracts`: parameter templates and interfaces
- `utils`: general utilities
